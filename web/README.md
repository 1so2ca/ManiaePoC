# Web（フロントエンド）

**広島県のバス運行情報を表示するウェブアプリケーション**

## 技術スタック
- **Elm** → JavaScript（Cloudflare Workers/Pages）
- **mdgriffith/elm-ui** - 型安全なUI構築

## Elmライブラリ構成

### 必須ライブラリ
- **elm/browser** - ナビゲーション・URL管理
- **elm/http** - HTTPリクエスト
- **elm/json** - JSON エンコード・デコード
- **mdgriffith/elm-ui** - 型安全なUI構築

## 機能

### メインページ
- バス停の現在の運行情報表示
- 到着予定時刻表示

### 検索ページ
- バス停検索
- お気に入り登録（localStorage）

## 開発方針（PoC）
- **セキュリティ対策：最小限**
- **オブザーバビリティ：最小限**
- **テスト：正常系のみ**
- エラーハンドリング最小限
- パフォーマンス最適化なし
